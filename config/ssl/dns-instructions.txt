
# DNS konfiguracija za domene

## Za www.frigosistemtodosijevic.me:
A record: @ → [IP ADRESA SERVERA]
CNAME record: www → www.frigosistemtodosijevic.me

## Za admin.me:
A record: @ → [IP ADRESA SERVERA]

## Cloudflare konfiguracija (preporučeno):
1. Dodajte domene u Cloudflare
2. Postavite DNS record-e
3. Omogućite "Full (strict)" SSL mode
4. Omogućite "Always Use HTTPS"
5. Omogućite "HSTS"

## Testiranje DNS-a:
nslookup www.frigosistemtodosijevic.me
nslookup admin.me
dig www.frigosistemtodosijevic.me A
dig admin.me A
